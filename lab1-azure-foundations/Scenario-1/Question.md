Problem Statement: You are a system administrator at a company that recently onboarded a new employee, John Doe, as a Software Developer. You need to create a user account for John in Azure Active Directory (Azure AD) to grant him access to organizational resources.

Steps:
Go to the Azure portal and navigate to Azure Active Directory > Users.
Click + New user.
Select Create user.
Enter the following information:
User Name: johndoe@yourorganization.onmicrosoft.com
Name: John Doe
Password: Auto-generated
Assign the Developer Role from Role Assignments.
Click Create.
